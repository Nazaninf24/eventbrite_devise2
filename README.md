# Eventbrite-devise

## Installation et lancement de l'application :

- Sur le Terminal :
1. ```$ bundle install```
2. ```$ rails db:drop```
3. ```$ rails db:create```
4. ```$ rails db:migrate```
5. ```$ rails db:seed```
6. ```$ rails server```

- Sur Navigator web :
http://localhost:3000
